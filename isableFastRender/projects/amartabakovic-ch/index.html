My personal website.

Web Development • 2021

For a while now, I wanted to create a personal website. I originally envisioned to develop it in plain HTML and CSS, then in Webpack and JS, afterwards in Vue.js and now I’ve decided once and for all to use something called Hugo.

Tech

The website is mainly powered by the static site generator Hugo, which compiles markdown into HTML files. Hugo allows for fast compilation times and lightweight websites.

The styling is done with Sass and Fontello is used for icons.

I deployed this site on Amazon AWS, because I wanted to learn a bit more about cloud and saw this as a good opportunity.

UI

I have decided to go with a simple two tone color scheme, consisting of the following colors:

  • White: #FFFFFF
  • Navy: #04173B

This site includes a dark mode. If you want to learn more about my implementation, you can read my blog post about it.

The fonts I used here are Playfair Display and Inter, both of which are available from Google Fonts.
