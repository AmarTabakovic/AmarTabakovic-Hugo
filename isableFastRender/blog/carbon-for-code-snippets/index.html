Carbon for code screenshots

Having trouble creating code snippets for e.g. Microsoft Word or social media? This is where Carbon comes into play.


May 9, 2021 • Miscellaneous • 1 min

Create and share beautiful images of your source code. Start typing or drop a file into the text area to get started.

Carbon is a web-based code snippet generator, which you can use to create screenshots of your source code. Carbon supports syntax highlighting for over 60 languages and is highly customizable.

Compared to creating a screenshot of the code editor or copying the code and pasting it into a Word OpenDocument object, this is much easier and aesthetically more pleasant (atleast in my opinion).
